# Champions for Progress

A Christian-based business directory and job board platform serving the community in Inglewood, California and beyond.

## Overview

Champions for Progress is a WordPress-based website that provides:
- **Business Directory**: Christian businesses can create profiles and connect with the community
- **Job Board**: Job listings with categorization (General vs Church/Ministry) and location filtering
- **Beyond Job Check**: Weekly prayer calls for job seekers (Wednesdays at 12:00 PM PST)
- **Membership System**: Recurring payment system for business listings

## Features

### Business Directory
- Member sign-up with recurring payments
- Business profile creation (name, description, contact info, photos)
- Multiple membership tiers (Basic, Premium, Platinum)
- Search and filtering by category and location
- Featured business listings

### Job Board
- Public job submission with admin moderation
- Job categories: General Employment and Church/Ministry
- Location filtering: LA Area vs Nationwide
- Job application system
- Job seeker accounts (free)

### Beyond Job Check (Prayer Call Section)
- Weekly prayer call information
- Zoom meeting integration
- Prayer request forms
- Testimonials and success stories

## Technology Stack

### Required Plugins

1. **Paid Memberships Pro Plus** ($297/year)
   - Handles membership subscriptions
   - Payment processing (Stripe & PayPal)
   - Recurring billing

2. **GeoDirectory Business** ($249/year)
   - Business directory functionality
   - Custom fields and categories
   - Search and filtering
   - Integration with membership system

3. **WP Job Manager Bundle** ($249/year)
   - Job board functionality
   - Job submission and moderation
   - Application management
   - Custom job fields

### Recommended Theme
- **Astra Pro** ($59/year) - Lightweight, customizable, great plugin compatibility

### Hosting Requirements
- **WP Engine** ($25/month) or similar managed WordPress hosting
- PHP 7.4 or higher
- MySQL 5.7 or higher
- SSL certificate (for payment processing)
